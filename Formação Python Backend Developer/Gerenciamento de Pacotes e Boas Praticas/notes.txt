Gerenciamento de Pacotes, Convenções e Boas Práticas Python


- Oque são pacotes em Python - 
Pacotes são módulos que podem ser instalados e utilizados nos programas python onde voce pode utilizar e baixar codigos feitos por outras pessaos para utilizar no seu projeto. 

- Oque é Pip -
Pip é o gerenciador de pacotes do python onde ele permitie instalar, atualizar e remover pacotes.

Exemplos Pip :

pip install (instalar)
pip uninstall (desistalar)
pip list (listar bibliotecas)
pip install --upgrade nome_do_pacote (Atualiza Pacote)
pip search termo_de_busca (Procurar Pacote)

- Ambientes Virtuais - 
python3 -m venv <nome do ambiente Virtual>
source <nome do ambiente Virtual>/bin/activate

- pipenv - 

pip install pipenv (instalar globalmente)
pipenv install 
pipenv uninstall
pipenv lock

- Oque é Poetry -
è uma ferramenta de ferenciammento de dependencias para python que permite declarar bibliotecas de que seus projetos dependem

pip install Poetry
poetry new myproject
cd myproject
poetry add numpy
poetry remove numpy
